## Process Improvements

### Triple-Enforcement Philosophy
For critical processes, use 3 discovery mechanisms:
1. **Belt**: Briefing section (first thing agents see)
2. **Suspenders**: Wrapper script (automation)
3. **Rope**: Session log checklist (visible tracking)

### Investigation-First Methodology
Every sprint phase starts with comprehensive investigation:
- Verify assumptions
- Find working examples
- Identify blockers
- Then execute with confidence

### No Can-Kicking
When ahead of schedule, complete more work instead of taking breaks

## Team Learnings Distilled

### What Works
1. Investigation-first prevents rework
2. Multi-agent "binocular vision" catches everything
3. Phase-boundary verification catches issues early
4. Evidence-based decisions build confidence
5. Strategic escalation prevents wasted effort

### Philosophy
- Assume work might be 75% complete
- Verify before implementing
- Test with real APIs early
- Document honest technical triage
- Build infrastructure when triggers met
