---

## Major Outcomes

### Decisions Made (8 Architectural Decisions)

1. **Lenses on Substrates** - Flexible analytical views, not rigid object taxonomy
2. **Four Substrates** - Entities, Places, Moments, Situations (but Situation is container, not peer)
3. **8 Spatial Dimensions = Perceptual Lenses** - One model, not two
4. **Three-Way Ownership** - Native (mind), Federated (senses), Synthetic (understanding)
5. **Six Metadata Dimensions** - Provenance, Relevance, Attention, Confidence, Relations, Journal
6. **Full Lifecycle with Composting** - 8 stages, multiple cycle shapes as metadata
7. **Two Journaling Layers** - Session Journal (audit) + Insight Journal (dreaming outputs)
8. **User Model as Native Object** - Principal deeply modeled, others progressively lighter

### Discoveries Through Sketching

| Discovery | Significance |
|-----------|--------------|
| **Situation as container** | Not a fourth parallel substrate—it's the frame that holds Moments |
| **"Noticed" not "Inferred"** | More human, less algorithmic language |
| **Core grammar found** | "Entities experience Moments in Places" |
| **Moment = bounded scene** | Theatrical unities: one time, one place, momentous |
| **Cycle shapes as metadata** | Circle, spiral, arc, cradle-to-grave, overlapping—shape matters |
| **Entity/Place as spectrum** | Same object can be either depending on grammatical role |
| **PPP for Moments** | Policy, Process, People (+AI) as contents of any Moment |
| **Δ = change** | All diagrams are snapshots of dynamic sequences |
| **Perception/Orientation/Judgment** | May be three cognitive faculties, not one lens list |
| **Die for Situation** | Happy accident—both "composed of moments" AND uncertainty |

## Key Insights

### On Process

- **Systematic decision-making works**: One question at a time, recommendation + rationale, decide and move
- **"Hold it open"**: Don't force precision before understanding emerges
- **Sketching discovers what thinking misses**: The hand found Situation-as-container, "Noticed" > "Inferred"
- **AI tools transcribe, humans discover**: Use AI for polish, humans for insight
- **Fractal elaboration**: At some point, remaining decisions become less load-bearing

### On the Object Model

- **Grammar-forward**: "Entities experience Moments in Places" is cleaner than category lists
- **Moment is the unit of meaningful work**: Not task, not document—the bounded scene where something happened
- **Piper is an Entity**: A player with identity, not just a lens on the world
- **The dolphin works**: Nonthreatening, curious, nonhuman intelligence—perfect brand embodiment

### On Convergence

- **No conflict with MVP priorities**: Current sprint builds engine, UX 2.0 designs appearance/feel
- **Strong alignment discovered**: Spatial dimensions = Perception, Canonical queries = Orientation, Ethical consensus = Judgment
- **Current UI is placeholder**: Redesign is additive, not competing
