**❗ CRITICAL FINDING: Theme Inconsistency**
- Primary web interfaces (home, standup) use **light theme**
- New interfaces (learning, personality) use **dark theme**
- Different blue shades: #3498db vs #007acc
- No unified design system evident

**Technology Stack**:
- Frontend: Vanilla JS (no React/Vue framework)
- CSS: Inline styles (no CSS framework, no Tailwind despite config file)
- Markdown: marked.js CDN
- Icons: Emoji-based (no icon system)
- State Management: None evident (vanilla JS fetch)
