| #7 | IntentClassifier missing DI | Fixed container chain |

### PM Intervention: Methodology Reset (17:44)

> "Wrong frame: 'Fix the bug so feature works.' Right frame: 'Learn what is not yet built correctly.'"

This redirected from rapid-fix mode to systematic investigation, leading to actual root cause discovery.

### Antipatterns Identified

1. **"Middleware-Route Contract Violation"**: Optional auth routes blocked
2. **"Mock the Integration Point"**: Tests mock the method they're testing
3. **"Construct Test Objects with Fake Data"**: Tests create objects real code never populates
4. **"Integration Tests That Don't Integrate"**: E2E files use heavy mocking
