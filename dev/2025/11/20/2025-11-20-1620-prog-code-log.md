## Lessons Learned

### What Went Well
1. **Archaeological approach paid off**: Tests were skipped not due to bugs, but outdated skip decorators
2. **Integration markers work correctly**: Auth tests properly isolate database behavior
3. **Auto-mock serves its purpose**: Protects unit tests without hiding integration issues
4. **Test recovery revealed production bugs**: 2 latent bugs fixed by running previously skipped tests

### Unexpected Discoveries
1. **All auth tests bypass auto-mock**: Investigation found they all use integration marker
2. **Mock expectations were wrong**: Tests assumed incorrect behavior (MONITOR vs PATROL)
3. **Timestamp field missing**: SpatialEvent needed optional timestamp for test compatibility

### Process Improvements
1. **Always check for integration markers** when investigating auto-mocks
2. **Run tests immediately after unskipping** to catch expectation mismatches
3. **Make new fields optional by default** for backward compatibility
