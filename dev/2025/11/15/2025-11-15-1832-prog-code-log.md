**For Sandbox Claude Code**:

**For Debugging Route/Mount Issues**:
1. Always do a **full clean restart** (`stop-piper.sh` then `start-piper.sh`)
2. Check for zombie processes: `ps aux | grep uvicorn`
3. Verify server is actually running on expected port before testing
4. Test with `curl -I` to get just headers for quick verification
5. If restarting doesn't work initially, try again - timing/cleanup issues possible
