## Why 6-Month Analysis Is Slow

**Bottleneck**: SemanticAnalyzer scanning hundreds of markdown files.

**Current approach**:
```python
# Scans EVERY file for EVERY concept
for file_path in all_markdown_files:  # 200+ files
    content = file_path.read_text()
    for term in key_concepts:  # 68 concepts
        matches = pattern.findall(content)  # O(n*m)
```

**Performance**: 200 files × 68 concepts = 13,600 pattern searches

**Optimization needed**:
1. Index files by date (skip files outside date range)
2. Cache file content
3. Parallelize file scanning
4. Use faster regex library (re2 instead of re)
