## Technical Insights Discovered

### 1. Breakthrough Signatures Are Non-Monolithic
Different breakthrough types have distinct signal patterns:
- **Implementation**: High ADR activity + refactoring
- **Discovery**: Concept emergence + parallel coordination
- **Mixed**: Can have multiple breakthrough types simultaneously

### 2. Git History Tracking Requires --follow
GitPython's `iter_commits(paths=...)` only sees commits at the current path. Files that were moved/renamed are invisible without `git log --follow --diff-filter=A`.

### 3. Cross-Context Validation Is Powerful
Concepts appearing in multiple contexts (ADR + code + omnibus) are more validated than single-context appearances. Validation score = present_contexts / total_contexts.

### 4. Temporal Clustering Reveals Patterns
Grouping signals by date reveals coordination patterns invisible in aggregate analysis. Nov 1 had 4 simultaneous signals; Nov 3 had 3.

### 5. Signal Convergence Indicates Confidence
Breakthroughs with signals from multiple analyzers are higher confidence than single-analyzer signals. Convergence bonus: (analyzers - 1) * 0.15
