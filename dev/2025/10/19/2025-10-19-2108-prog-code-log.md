## The Challenge: Import Issues

### Initial Discovery

Found existing test file `tests/api/test_standup_api.py` (18888 bytes, 20 tests already written!) but pytest couldn't import it:

```
ModuleNotFoundError: No module named 'web.api'
```

**The Problem**:
- Test used direct router import: `from web.api.routes.standup import router`
- Worked fine in direct Python execution
- Failed specifically in pytest

**STOP Condition Used**: After ~30 minutes debugging, used STOP Condition #17: "Bash + JSON not working" (adapted for pytest import issues)

### PM's Solution

PM provided solution document: `/Users/xian/Development/piper-morgan/dev/active/code-task-6-pytest-import-solution.md`

**Key Guidance**:
- Don't import router directly
- Use FastAPI `TestClient` pattern instead
- Import from `web.app`, not router modules
- Use context manager with session-scoped fixture for lifespan

**Example**:
```python
# ❌ WRONG - Direct router import
from web.api.routes.standup import router as standup_router

# ✅ CORRECT - TestClient pattern
from fastapi.testclient import TestClient
from web.app import app

@pytest.fixture(scope="module")
def client():
    with TestClient(app) as test_client:
        yield test_client
```

## Key Technical Decisions

### TestClient Pattern

**Choice**: Use FastAPI TestClient with lifespan support

**Why**:
- Triggers FastAPI lifespan events (startup/shutdown)
- Tests actual HTTP behavior
- No import path issues
- Standard FastAPI testing pattern

**Implementation**:
```python
@pytest.fixture(scope="module")
def client():
    with TestClient(app) as test_client:
        yield test_client
```

### Session-Scoped Fixture

**Choice**: `scope="module"` for client fixture

**Why**:
- Reuses same app instance across all tests
- Faster test execution
- Triggers lifespan once per module
- Matches real server behavior

### Authentication Strategy

**Choice**: JWT fixtures with real token generation

**Why**:
- Tests actual auth flow
- Verifies JWT service integration
- Matches production authentication
- No mocking of auth layer
