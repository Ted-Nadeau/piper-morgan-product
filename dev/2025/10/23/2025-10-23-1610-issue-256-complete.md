### Technical Details

**Smart Browser Detection**:

```python
def should_open_browser() -> bool:
    # Don't open if explicitly disabled
    if args.no_browser:
        return False

    # Don't open if no display (CI/CD, SSH)
    if not os.environ.get("DISPLAY") and sys.platform != "darwin":
        return False

    # Don't open for commands (setup, status, etc.)
    if args.command:
        return False

    return True
```

**Delayed Opening**:

- 2-second delay to ensure server is ready
- Async task doesn't block server startup
- Graceful error handling for browser failures

**Cross-Platform Support**:

- **macOS**: Always attempts to open (no DISPLAY check needed)
- **Linux**: Checks for DISPLAY environment variable
- **Windows**: Standard webbrowser module support
