### Technical Implementation

**Policy Configuration** (`config/rotation_policy.yaml`):

```yaml
rotation_policies:
  default:
    max_age_days: 90
    warning_days: [60, 75, 85]
    critical_days: 88

  provider_overrides:
    openai:
      max_age_days: 60 # More frequent for high-value keys
    github:
      max_age_days: 180 # Less frequent for personal tokens
```

**Key Age Calculation**:

```python
def _calculate_key_age(self, key) -> int:
    # Use rotated_at if available, otherwise created_at
    last_update = key.rotated_at or key.created_at
    return (datetime.utcnow() - last_update).days
```

**Reminder Severity Logic**:

- **Critical**: Age >= critical_days (immediate action)
- **Warning**: Age in warning_days list (plan rotation)
- **Info**: Age >= (first_warning - 7) (awareness)
