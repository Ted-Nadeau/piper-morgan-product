### Query Pattern

- Reasoning chains from causal edges
- Intent hints improve classification
- ~50%+ token savings potential through focused context
- Cost-effective: Context <500 tokens

### Integration Flow
1. Intent classification triggered

3. Intent hints extracted from context
4. Hints guide LLM classification
5. Result includes graph reasoning
6. Event emitted for learning loop
