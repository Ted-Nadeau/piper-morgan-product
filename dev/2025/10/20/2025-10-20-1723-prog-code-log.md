### Created: web/assets/learning-dashboard.html (939 lines)

**Architecture Decision**: Single self-contained HTML file
- Rationale: Zero build step, no dependencies, fast deployment
- Vanilla JavaScript (ES6+)
- Embedded CSS and JS in one file
- Total: ~200 HTML + ~400 CSS + ~340 JS

### Dashboard Structure (5 Cards)

#### 1. Learning Status Card
```html
- Real-time status badge (Enabled/Disabled/Loading)
- One-click toggle button with confirmation
- Visual indicators: 🟢 green (enabled), 🔴 red (disabled), ⚪ gray (loading)
```

#### 2. Learning Metrics Card
```html
- Total Patterns Learned (number)
- Active Preferences (number)
- Success Rate (percentage)
- Last Updated (timestamp)
- Auto-refresh every 30 seconds
```

#### 3. Pattern Distribution Card
```html
- Horizontal bar chart visualization
- Pattern types: workflow, query, issue_similarity, ui_preference, filter_preference
- Percentage distribution display
- Empty state: "No patterns learned yet"
```

#### 4. Privacy Settings Card
```html
- 5 toggle switches with auto-save:
  1. Share patterns across features (OFF)
  2. Allow pattern suggestions (OFF)
  3. Enable cross-user learning (OFF)
  4. Store detailed metadata (OFF)
  5. Analytics participation (OFF)
- Conservative defaults (privacy-first)
```

#### 5. Data Management Card
```html
- Export Data to JSON (with timestamp filename)
- Clear Data with options:
  * All data (double confirmation required)
  * Patterns only
  * Preferences only
```

### JavaScript Controller (16 Functions)

**Core Functions**:
```javascript
async loadDashboard()           // Orchestrator with Promise.all()
async loadLearningStatus()      // Status badge and toggle
async loadMetrics()             // Analytics display
function updatePatternDistribution(distribution)  // Bar chart
async loadPrivacySettings()     // Privacy toggles
async toggleLearning()          // Enable/disable with confirmation
async savePrivacySettings()     // Auto-save on change
async exportData()              // JSON download via Blob API
async confirmClearData(dataType)  // Double confirmation for "all"
async refreshDashboard()        // Manual refresh
function startAutoRefresh()     // 30s timer
function updateTimestamp()      // Last refresh time
function showError(message)     // Error banner
function showSuccess(message)   // Success banner
function hideMessage(type)      // Auto-dismiss messages
```

**Event Handlers**:
```javascript
window.addEventListener('DOMContentLoaded', async () => {
    await loadDashboard();
    updateTimestamp();
    startAutoRefresh();
});

// Keyboard shortcuts
Ctrl+R: Manual refresh (preventDefault browser reload)
Ctrl+E: Export data to JSON
```

### API Integration (8 Endpoints)

```javascript
const API_BASE = '/api/v1/learning';
const USER_ID = 'current_user';  // Configurable
const AUTO_REFRESH_INTERVAL = 30000;  // 30 seconds
```

**Endpoints Integrated**:
1. `GET /analytics` → Metrics and pattern distribution
2. `GET /controls/learning/status?user_id=` → Status badge
3. `POST /controls/learning/enable` → Enable button
4. `POST /controls/learning/disable` → Disable button
5. `GET /controls/privacy/settings?user_id=` → Privacy toggles
6. `POST /controls/privacy/settings` → Privacy save
7. `GET /controls/export?user_id=&format=json` → Export download
8. `DELETE /controls/data/clear?user_id=&data_type=` → Data clearing

**Error Handling Pattern**:
```javascript
try {
    const response = await fetch(url, options);
    if (!response.ok) throw new Error(`HTTP ${response.status}`);
    const data = await response.json();
    // Process data
    showSuccess('Operation successful');
} catch (error) {
    console.error('Error:', error);
    showError('Operation failed: ' + error.message);
}
```
