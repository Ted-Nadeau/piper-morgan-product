### 2. Repository Sorting Verification

From `TodoRepository.get_todos_by_owner()`:
```python
query = query.order_by(
    TodoDB.priority.desc(),
    TodoDB.due_date.asc().nulls_last(),
    TodoDB.created_at.desc()
).limit(limit)
```

This means the first todo returned is always the highest priority (urgent > high > medium > low), with secondary sorting by due date and creation time.
