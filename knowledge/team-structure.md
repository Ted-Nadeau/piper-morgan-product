## Communication Dynamics

### Session Patterns
- **Chief of Staff**: Weekly strategic reviews, Weekly Ship preparation
- **Chief Architect**: Pattern decisions, ADR creation, strategic technical planning
- **Lead Developer**: Daily execution, agent coordination, code review
- **Communications Director**: Editorial strategy, content creation

### Coordination Channels
- **GitHub Issues**: Technical progress tracking
- **Session Logs**: Detailed execution evidence
- **Omnibus Logs**: Daily synthesis for leadership
- **Weekly Ships**: Public progress summary

### Escalation Paths
- Technical blockers → Chief Architect
- Resource/priority → PM (xian)
- Process/methodology → CIO (xian acting)
- Agent coordination → HOSR (when created) / Chief of Staff (interim)
