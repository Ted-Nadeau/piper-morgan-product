# Piper Morgan - Technology Stack

## Core Technologies
- **Python**: 3.11+ required
- **FastAPI**: Web framework for API and web interface
- **Jinja2**: Template engine for HTML rendering
- **PostgreSQL**: Primary database (port 5433)
- **Redis**: Caching layer
- **Docker**: Containerization and development environment

## Key Libraries & Frameworks
- **Pydantic**: Data validation and settings management
- **SQLAlchemy**: Database ORM
- **Alembic**: Database migrations
- **pytest**: Testing framework with asyncio support
- **uvicorn**: ASGI server
- **ChromaDB**: Vector database for embeddings

## Development Tools
- **Black**: Code formatting (line length 100)
- **isort**: Import sorting
- **flake8**: Linting
- **pre-commit**: Git hooks for code quality
- **pytest**: Testing with coverage reporting

## Architecture Patterns
- **Plugin System**: Modular integrations with auto-registration
- **Intent Classification**: LLM-based routing system
- **Domain-Driven Design**: Clear separation of concerns
- **Repository Pattern**: Data access abstraction
- **MCP (Model Context Protocol)**: For external integrations

## External Integrations
- **GitHub**: Repository and issue management
- **Slack**: Workspace communication
- **Notion**: Knowledge management
- **Google Calendar**: Schedule management
