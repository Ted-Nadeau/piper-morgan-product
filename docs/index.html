What is Piper Morgan?

Piper Morgan is an AI-powered Product Management Assistant that evolves from automating routine PM tasks to providing strategic insights. It's built with domain-driven design where PM concepts (Product, Feature, Intent, WorkItem) drive the technical architecture.

Current Status

Phase 3 Complete - ConversationManager implementation with anaphoric reference resolution achieving 100% accuracy and 2.33ms latency (65x faster than target).

Key Features

  • Intent Classification: Understands PM requests and routes them appropriately
  • GitHub Integration: Create and manage issues directly from conversations
  • Slack Integration: Spatial metaphor for team collaboration
  • Document Analysis: Extract insights from product documents
  • Workflow Orchestration: Manage complex multi-step PM processes
  • Context Awareness: 10-turn conversation memory with reference resolution

Architecture Highlights

  • Domain-Driven Design: PM concepts drive the technical implementation
  • CQRS-lite Pattern: Separated read and write operations
  • Graceful Degradation: Circuit breakers and fallback strategies
  • Multi-LLM Strategy: Claude for reasoning, OpenAI for embeddings
  • AsyncSessionFactory: Standardized async database session management

Getting Started

# Clone the repository
git clone https://github.com/mediajunkie/piper-morgan-product.git
cd piper-morgan-product

# Set up Python environment (requires Python 3.11)
python3.11 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Start infrastructure
docker-compose up -d

# Initialize database
python scripts/init_db.py

# Run the API server
python main.py
