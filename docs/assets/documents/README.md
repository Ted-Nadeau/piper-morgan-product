# Document Assets

## Overview

This directory contains document assets, including PDFs, presentations, specifications, and other document files that support the Piper Morgan project documentation.

## Organization

This directory is organized into 2 subdirectories, each containing specific types of visual assets and supporting materials.

## Asset Guidelines

When working with assets in this directory:
- **Naming**: Use descriptive, consistent naming conventions
- **Formats**: Maintain appropriate file formats for different use cases
- **Organization**: Keep assets organized by topic or system area
- **Documentation**: Include descriptions for complex diagrams or documents
