## Capturing Screenshots

To capture new screenshots:
1. Start fresh Piper Morgan instance (or reset setup state)
2. Navigate through setup wizard
3. Capture at 1440x900 or similar 16:10 ratio
4. Save as PNG with descriptive filename
5. Update this README if adding new screenshots
