Multi-Project Intelligence

🎯 PM-009: Sophisticated Context Resolution

Challenge: PMs work across multiple projects with implicit context
Solution: Hierarchical resolution with AI inference
1. Explicit project_id (always wins) 2. Session memory (if confirmed) 3. LLM inference from message content 4. Default project fallback
Example: "Create a bug ticket for the mobile login crash"
→ AI infers "Mobile App" project from context
→ Remembers choice for session
→ Asks for confirmation if ambiguous

✅ What Works

  • Graceful ambiguity handling
  • Integration-specific configuration
