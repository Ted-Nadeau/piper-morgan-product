# Methodologies

**Purpose**: Systematic approaches to development, project management, and team collaboration

## Organization

### **Established Methodologies** ([./established/](./established/))

Proven methodologies and approaches that we consciously adopted and applied.

**Key Characteristics**:

- Well-documented and widely used
- Include citations and references to original sources
- Consciously chosen and implemented
- Examples: Three-Phase Evolution, Session Management, Agile Development

### **Emergent Methodologies** ([./emergent/](./emergent/))

Methodologies that emerged through our development process and team collaboration.

**Key Characteristics**:

- Discovered through real-world development challenges
- Include evolution history and problem-solution mapping
- Emerged from team collaboration and process optimization
- Examples: Pattern-Driven Development, AI-Assisted Development Methodology

## Methodology Categories

### **Development Methodologies**

- Software development process approaches
- Code organization and workflow patterns
- Quality assurance and testing methodologies

### **Project Management Methodologies**

- Project planning and execution approaches
- Team coordination and communication patterns
- Stakeholder management methodologies

### **Collaboration Methodologies**

- Team collaboration and communication approaches
- Human-AI collaboration patterns
- Knowledge sharing and transfer methodologies

### **Evolution Methodologies**

- System and process evolution approaches
- Continuous improvement patterns
- Adaptation and learning methodologies

## Usage Guidelines

1. **Choose by Context**: Select methodologies based on project context and team needs
2. **Adapt and Evolve**: Modify methodologies based on team dynamics and project requirements
3. **Document Processes**: Use methodologies to structure and document development processes
4. **Measure Effectiveness**: Track methodology effectiveness and outcomes
5. **Share Learnings**: Contribute methodology improvements and discoveries
